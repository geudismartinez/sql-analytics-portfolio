## 📋 Skills Demonstrated

### 🔍 Phase 0 — Database Exploration
- Exploring tables and columns with `INFORMATION_SCHEMA`
- Counting rows, checking NULL values, understanding date ranges
- Understanding relationships between tables

### 🟢 Phase 1 — Foundations
- `SELECT`, `WHERE`, `ORDER BY`, `DISTINCT`
- `GROUP BY`, `HAVING`, aggregate functions
- `INNER JOIN`, `LEFT JOIN` across multiple tables

### 🟡 Phase 2 — Intermediate
- Simple subqueries (`IN`, `NOT IN`, scalar)
- **Correlated subqueries** — referencing the outer query
- **Derived tables** — subqueries in the `FROM` clause
- `CASE WHEN` — simple, nested, and inside aggregates
- Manual pivot technique with `CASE` inside `SUM`

### 🔵 Phase 3 — Advanced
- **CTEs** — simple, multiple, and chained
- **Recursive CTEs** — employee hierarchy tree
- **Window Functions:**
  - `ROW_NUMBER`, `RANK`, `DENSE_RANK`
  - `LAG`, `LEAD` — period over period comparisons
  - `NTILE` — customer segmentation
  - `FIRST_VALUE`, `LAST_VALUE`
  - `PERCENT_RANK`, `CUME_DIST`
  - Running totals and moving averages (`ROWS BETWEEN`)

### 🔴 Phase 4 — Professional
- **RFM Analysis** — Recency, Frequency, Monetary customer segmentation
- **Year-over-Year growth** analysis
- **ABC / Pareto classification** — product revenue analysis
- `TRY...CATCH` — error handling
- Transactions with `ROLLBACK` protection
- Data validation patterns
- Executive dashboard query — full business summary

## 💡 Key Business Questions Answered

- Who are our top customers by revenue and how do we segment them?
- Which products generate 80% of our revenue? (Pareto/ABC Analysis)
- How is our business growing month over month and year over year?
- Which employees are our top performers?
- What is the full customer lifecycle from first to last order?
- Which products have never been ordered?
- How do we classify customers by behavior? (RFM Analysis)

## 🚀 How to Run This Project

1. Install [SQL Server Express](https://www.microsoft.com/en-us/sql-server/sql-server-downloads) (free)
2. Install [SSMS](https://learn.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms) (free)
3. Download and restore the [Northwind database](https://github.com/microsoft/sql-server-samples/tree/master/samples/databases/northwind-pubs)
4. Open `SQL_Analytics_Portfolio_Northwind.sql` in SSMS
5. Run each phase step by step
